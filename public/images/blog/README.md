The `speed-*` and `*-hero` charts are generated from the artificial dataset in
`/public/data/speed-journey.csv` (see the 95→130 post for context) and are
styled to the site palette.

`heroImage` in a post's frontmatter is optional — delete the line if a post has
no hero, and the layouts adapt automatically.
